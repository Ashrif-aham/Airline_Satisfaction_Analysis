# ✈️ Airline Passenger Satisfaction Analysis

## 📌 Project Overview

This project analyzes airline passenger satisfaction using machine learning techniques. The goal is to identify key factors that influence customer satisfaction and build a predictive model to classify passengers as satisfied or dissatisfied.

---

## 📊 Dataset Features

The dataset includes the following features:

* Customer Details: Gender, Customer Type, Age
* Travel Information: Flight Distance, Type of Travel, Class
* Service Ratings:

  * Seat Comfort
  * Food and Drink
  * Inflight Entertainment
  * On-board Service
  * Cleanliness
  * Online Support
* Delay Information:

  * Departure Delay
  * Arrival Delay

---

## 🔍 Exploratory Data Analysis (EDA)

* Distribution of satisfaction levels
* Analysis of categorical features vs satisfaction
* Histogram and boxplots for numerical features
* Correlation heatmap

---

## ⚙️ Feature Engineering

* Converted categorical variables into numerical format
* Created new features:

  * **Total Delay**
  * **Service Score**
  * **Online Score**
* Handled missing values and duplicates

---

## 🤖 Model Building

* Model Used: Logistic Regression
* Data Split: 80% Training, 20% Testing
* Feature Scaling applied using StandardScaler

---

## 📈 Model Evaluation

* Accuracy Score
* Classification Report (Precision, Recall, F1-score)
* Confusion Matrix Visualization

---

## 🎯 Results

* Achieved an accuracy of approximately **85%–90%**
* Service-related features showed strong impact on satisfaction

---

## 🛠️ Technologies Used

## 📌 Key Insights

* Better onboard services lead to higher satisfaction
* Delays negatively impact customer experience
* Online services play a significant role
